# Taller 3 - Layouts con Bootstrap 5

Este proyecto es una aplicación web responsiva desarrollada como parte del proceso de formación en **ADSO**. El objetivo principal es poner en práctica el diseño de interfaces web dinámicas, limpias y optimizadas utilizando los componentes nativos de Bootstrap.

## 🚀 Características del Proyecto
* **Barra de Navegación Avanzada:** Navbar fija con distribución lógica de menús desplegables (`Dropdowns`) y accesos directos alineados.
* **Componentes Dinámicos:** Integración de un carrusel de imágenes adaptativo para mejorar la experiencia visual.
* **Secciones Responsivas:** Grid de Bootstrap optimizado para visualización correcta en dispositivos móviles, tablets y pantallas de escritorio.
* **Formularios y Tablas:** Estructuras de datos organizadas y componentes de inicio de sesión estilizados sin hojas de CSS personalizadas pesadas.

## 🛠️ Tecnologías Utilizadas
* **HTML5** - Estructura semántica del sitio.
* **Bootstrap 5.3.3** - Framework de estilos y maquetación responsiva.
* **Bootstrap Icons 1.13.1** - Librería de iconos vectoriales.
